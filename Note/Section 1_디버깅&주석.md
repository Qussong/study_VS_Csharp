## 디버깅 콘솔 정리
- 도구 → 옵션 → 디버깅 → 일반 → "디버깅이 중지되면 자동으로 콘솔 닫기" 선택 → 확인
- 위 옵션을 설정하면 코드 실행시 바로 콘솔 출력창이 종료된다.
- 맨 아래 `Console.ReadKey();` 코드 추가한다. 콘솔 출력창이 깔끔해지며, 추가 입력이 있을때까지 출력창이 종료되지 않고 대기한다.

## 용어 정리
```csharp
Console.WriteLine("Hello, World");
// Cosole : 클래스
// WriteLine() : 메서드
// "Hello, World" : 인수, 문자열(string)
```
- `클래스`:</br>
데이터(속성)와 기능(메서드)을 하나의 틀로 묶어 놓은 구조
- `메서드`:</br>
클래스 내에서 특정 동작이나 기능을 수행하는 코드 블록
- `인수`:</br>
메서드가 실행될 때 필요한 데이터를 전달하기 위한 매개변수
```csharp
string message = "안녕하세요";
```
- `문자열(string)`:</br>
텍스트 데이터를 저장하고 조작하는 데 사용되는 데이터 형식
- `선언`:</br>
변수, 메서드, 클래스 등 프로그래밍 요소를 정의하고 이름을 지정하는 과정
- `디버깅`:</br>
프로그램에서 발생하는 오류나 버그를 찾아내고 수정하는 과정

## 주석
```csharp
// 단일 행 주석

/*
    다중 행 주석
*/
```
- `단일 행 주석`:</br>
한 줄로 작성된 주석을 의미하며, 코드의 일부를 설명하거나 설명을 추가하는 용도로 사용
- `다중 행 주석`:</br>
여러 줄에 걸쳐 작성할 때 사용
- `컴파일`:</br>
소스 코드를 컴퓨터가 실행할 수 있는 기계어로 변환하는 과정
